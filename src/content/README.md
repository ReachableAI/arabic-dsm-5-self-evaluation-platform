# Content & Questions

This directory contains static content, assessment questions, and educational resources for the Arabic DSM-5-TR Self-Evaluation Platform.

## Structure

- `questions/` - Assessment question banks organized by category
- `resources/` - Educational resources and help content
- `disclaimers/` - Legal disclaimers and safety messaging
- `crisis/` - Crisis intervention resources

## Guidelines

- All content should be in Modern Standard Arabic (MSA)
- Include mental health professional review notes
- Follow safety protocols for sensitive content
- Maintain version control for question updates
- Document source references (DSM-5-TR criteria)
